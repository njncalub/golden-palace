# PAO Directory — Format Template

This directory holds the **PAO library** (Person-Action-Object) for numbers 00–99. Each entry turns a two-digit number into a memorable scene: a **Person** doing an **Action** with an **Object**. Golden Palace prioritizes **Biblical characters** so your palace stays "Golden" (Holy of Holies imagery).

## PAO Entry Format

```markdown
## [Number], Person: [Name]

**Number:** [00-99]
**Person:** [Name]
**Action:** [Vivid memorable action]
**Object:** [Associated object]
**Source:** [Biblical / Modern]
**Scripture connection:** [Relevant verse if Biblical figure]
**Memory hook:** [Why this person/action/object is memorable]
```

## File Naming Convention

Format: `[two-digit]-[person-slug].md`

- Examples: `01-moses.md`, `02-david.md`, `40-moses-parts-sea.md`.
- Place Biblical figures in **pao/biblical/**; modern teachers (pastors, theologians) in **pao/modern/** when no clear Biblical peg fits.

## Why Biblical First ("Golden" Pegs)

Use **characters from the Bible itself** wherever possible — e.g. **Moses with his staff** (1 or 40), **David slinging a stone** (2), **Esther with her crown**. That way the imagery encoding verse references is Scripture-saturated and your mind palace remains a Holy of Holies, not a random gallery. Only use modern teachers for numbers where no natural Biblical peg exists.

## Guidelines

- **Visual, active, emotional** — You must be able to see the scene clearly.
- **Philippians 4:8** — All images true, noble, right, pure, lovely, admirable.
- **Hebrew Major System** — For numbers without an obvious Biblical link, use the digit→Hebrew letter mapping to generate ideas (e.g. 3 = Gimel = grace → a grace scene).
