# PAO Library

Person-Action-Object entries for numbers 00–99. Each number gets a memorable scene so you can encode verse references (e.g. John 3:16 → 03 + 16) as images. **Biblical figures first** — Moses with his staff, David with his sling — so your palace stays Golden.

## Format and naming

- See [FORMAT.md](/pao/FORMAT.md) for the PAO entry template, file naming, and "Why Biblical first."
- Files: `[two-digit]-[person-slug].md` in **pao/biblical/** or **pao/modern/**.
