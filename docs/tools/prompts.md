# Prompts & Study Tools

Golden Palace gives you the **memory system**; this page gives you **prompts and tools** to study verses and run devotions _within_ that system. This is not a devotion guide — it is a set of copy-paste prompts and tool notes so you can use ChatGPT (or your favorite LLM) and apps effectively.

## Prompts to Use With an LLM

Copy, paste, and fill in the bracketed parts.

### Study a verse for memorization

```
I'm memorizing [e.g. Psalm 23:1] with the Golden Palace method (Hebrew Major System, PAO, mind palace). Give me:
1. The verse in 2–3 translations.
2. 1–2 key Hebrew or Greek words with pictographic or root meaning.
3. One vivid image that captures the main idea for a mind palace (Philippians 4:8 compliant).
4. A one-sentence theological hook.
Keep it concise.
```

### Choose study tools

```
I want to study Scripture for memorization and devotion. Compare Versify, Enduring Word, Blue Letter Bible (and any others you recommend) for: verse-by-verse explanation, word studies, and use during a 10-minute devotion. Which would you use for what?
```

### Structure a short devotion

```
I have 15 minutes for devotion. I use mind palace + PAO + spaced repetition. Suggest a simple flow: which verse(s) to review, whether to add one new verse, and one question to sit with. Don't write the devotion — give me a structure.
```

### Ask for an analogy

```
Explain [concept from this verse] with a simple analogy that I can turn into a mental image for memory and meditation.
```

## Study Tools (Brief Notes)

- **Enduring Word** — Verse-by-verse commentary, often pastoral and accessible. Good for "what does this mean?" before encoding.
- **Blue Letter Bible** — Strong’s numbers, Hebrew/Greek interlinear, word studies. Good for key words and pictographic hooks.
- **Versify** — Check current offering; often app-based verse engagement (reading, listening, repetition). Use for exposure and review, not replacement for palace encoding.
- **Bible Project** — Overviews, themes, visuals. Excellent for book-level storyline and "where does this fit?" (see [Books](/books/)).

Use these to _fill_ your palace with good study; the palace is where it gets stored and recalled.
