# Visual Meditations

Guided visualizations for specific verses so you can _feel_ the truth, not just know it. Each meditation uses a central image and five-senses prompts (see, hear, touch, smell/taste) and takes about 3–5 minutes.

## Format and naming

- See [FORMAT.md](/meditations/FORMAT.md) for the meditation guide template and file-naming convention.
- File names match [verse entries](/verses/) where possible (e.g. `ephesians-3-18-19.md`).
