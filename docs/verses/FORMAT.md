# Verses Directory — Format Template

This directory contains individual verse memory entries. Each file documents one verse (or short range) for memorization using the Golden Palace system.

## Verse Entry Format

```markdown
## [Book] [Chapter]:[Verse]

**Reference:** [Book] [Chapter]:[Verse]
**Text:** "[Full verse text]"
**Translation:** [Translation used]

### Hebrew/Greek Key Words

- [Word] ([Original]) — [Pictographic meaning if Hebrew]

### Gematria

- [Any relevant numerical connections]

### Memory Image

**Chapter encoding:** [digit] = [Hebrew letter] = [image]
**Verse encoding:** [digit(s)] = [Hebrew letter(s)] = [image]
**Combined PAO scene:** [Full vivid scene description]
**Palace location:** [Which palace + which anchor point]

### Theological Note

[1-3 sentences on what this verse means in context]

### Cross References

- [Related verses]
```

## File Naming Convention

Format: `[book]-[chapter]-[verse].md` or `[book]-[chapter]-[verse-start]-[verse-end].md`

- Use **lowercase**, **hyphenated** book names.
- Examples: `john-3-16.md`, `psalm-23-1.md`, `ephesians-3-18-19.md`, `genesis-1-1.md`.
- Multi-word books: `1-kings.md`, `song-of-solomon.md`, `revelation.md`.

## Guidelines

- **Philippians 4:8** — Images must be true, noble, right, pure, lovely, admirable.
- **Bridge to meditation** — If a [visual meditation](/meditations/) exists for this verse, link to it in the Theological Note or add a "See also" line.
