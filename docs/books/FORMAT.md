# Books Directory — Format Template

This directory contains **one page per book** of the Christian Bible. Each page gives a TLDR, overarching storyline, pegs for memorizing the book, highlights, how it fits with the previous/next book, place in the overarching biblical story, and interesting facts (including prophecies affirmed in later books).

## Book Page Format

```markdown
# [Book Name]

**TLDR:** [One to three sentences: what this book is about and why it matters.]

## Overarching Storyline

[Main narrative or argument. Optional: link or embed Bible Project overview.]

## Pegs for Memorizing This Book

[Memory hooks for the book as a whole: person (e.g. Moses for Exodus), key image (tabernacle, burning bush), number (40, 12), or short phrase. So "which book?" and "where does this verse live?" stay clear.]

## Highlights

[Key verses or passages; link to verse entries in /verses/ and meditations where they exist.]

## Before and After

[How this book differs from the previous and next book in the canon.]

## Place in the Overarching Biblical Story

[Where it sits in Creation → Fall → Israel → Christ → Church → New creation. One short paragraph.]

## Interesting Facts

[Quotes, prophecies fulfilled in later books, historical/cultural notes, gematria or Hebrew connections where relevant.]
```

## File Naming Convention

Format: `[two-digit]-[book-slug].md`

- Use **two-digit** prefix (01–66) in **standard Protestant 66-book order** so files sort in canonical order.
- **Lowercase**, hyphenated book slug for multi-word books.
- Examples: `01-genesis.md`, `02-exodus.md`, `11-1-kings.md`, `43-john.md`, `66-revelation.md`.
