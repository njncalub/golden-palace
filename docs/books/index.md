# Books of the Bible

One page per book: TLDR, storyline, pegs for memorizing the book, highlights, before/after, place in the overarching story, and interesting facts. Use these for big-picture context and book-level memory pegs.

## Format and naming

- See [FORMAT.md](/books/FORMAT.md) for the book page template and file-naming convention.
- File names: `[two-digit]-[book-slug].md` (e.g. `01-genesis.md`, `43-john.md`, `66-revelation.md`).
